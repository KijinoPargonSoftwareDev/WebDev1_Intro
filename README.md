# WebDev1_Intro

## Part 1

- Clone your repository with VS Code, or open it with CodeSpaces.
- Create an `index.html` file with the following:
	- The doctype declaration, head, and body elements following HTML standard
	- An H1 element with your name
	- Pick 3 or more of the following questions/topics and use an H1 to introduce each, as well as Paragraph and Image elements to answer the questions:
		- Favorite culture you're not native to?
		- Two or more historical figures that inspire you?
		- Using the language of flowers, what 3 flowers would make a bouquet representing you?
			- [https://www.almanac.com/flower-meanings-language-flowers](https://www.almanac.com/flower-meanings-language-flowers)
		- Finish the sentence: "I feel most creative when I.."
		- Build a playlist that represents you.
- Put any other info you want on this site.
- Push your changes when you're done.
- Place a comment on the myTech assignment when you've completed it and want it checked.

## Part 2

- Open the site you created for the Intro HTML Site assignment.
- Add CSS to the site to change the following:
  - The background (set it to either a solid color, gradient, or image)
  - The font color, size, or font itself.
  - Create style rules for at least two classes and use those classes on one or more elements.
  - Anything else you want to change.
  - I encourage you to take a look at W3Schools CSS Reference:
	  - https://www.w3schools.com/cssref/index.php
  - Make it pretty :)
- Make any other changes you want.
- Push your changes when you're done.
- Place a comment on the myTech assignment when you've completed it and want it checked.
